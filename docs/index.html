PassPhotoCheck

Lokale biometrische Passfoto-Analyse nach strengen DE/BMI und ICAO/ISO Standards. 100% Offline für maximale Privatsphäre.

Code auf GitHub ansehen Zurück zur Übersicht

Geometrie-Checks

Präzise Vermessung der biometrischen Merkmale: Kopfgröße, Augenabstand und Kinnposition werden automatisch an einer idealen Positionierungsschablone ausgerichtet und validiert.

Qualitäts-Prüfung

Umfassende Prüfung der Bildqualität. Erkennt sofort Unterbelichtung, starken Kontrast, Schattenwurf im Gesicht und mangelnde Schärfe, um die Akzeptanz bei Behörden zu sichern.

100% Offline & Privat

Als portable Windows Desktop-Anwendung konzipiert, benötigt PassPhotoCheck keine Cloud-Abhängigkeiten. Alle Daten und Fotos verbleiben lokal auf Ihrem Gerät.
